Movie and User API Documentation

This documentation provides detailed information about the API endpoints available in the Movie and User API. It includes the HTTP methods, endpoints, parameters, request bodies, and descriptions of each API function. Additionally, it outlines the Mongoose logic used in the backend for database operations and the expected format of the API responses.

Mongoose Logic

Method Endpoint Parameters Auth Request Body Description Mongoose Logic Expected Response
GET /movies no Returns a list of all movies. Movies.find({}) [{ movieObject1 }, { movieObject2 }, ...]
GET /movies/:title Title (string) no Returns data about a single movie. Movies.findOne({ title: req.params.title }) { movieObject }
PUT /movies/:title Title (string) yes update a movie by title Movies.findOneAndUpdate({ Title: req.params.Title }) { movieObject }
GET /movies/genres Name (string) no Returns a list of all genres. Movies.distinct("Genre.Name") { movieObject }
GET /movies/genres/:genreName Name (string) no Returns data about a genre. Movies.findone({ "Genre.Name": req.params.genreName }) { movieObject }
GET /directors Name (string) no Returns a list of all directors. Movies.distinct("Director.Name") { movieObject }
GET /directors/:directorName Name (string) no Returns data about a director. Movies.findOne({"Director.Name": req.params.directorName }), { movieObject }
POST /users no { username, password, email, birthday } Allows new users to register. Users.create(req.body) { success: true, message: "User registered successfully" }
POST /login no { username, password } Allows new users to generate JWT token. passport.authenticate { return res.json({ user, token }) }
PUT /users/:Username Username (string) { username, password, email, birthday } Allows users to update their user info. Users.findOneAndUpdate({ Username: req.params.Username }) { success: true, message: "User updated successfully", updatedUser: { updatedUserInfo } }
POST /users/:username/movies/:MovieID Username (string), MovieID (string) yes Allows users to add a movie to their favorites. updatedUser = await Users.findOneAndUpdate({ _id: req.params.userId}) { success: true, message: "Movie added to favorites successfully" }
DELETE /users/:Username/movies/:MovieID Username (string), MovieID (string) yes Allows users to remove a movie from their favorites. updatedUser = await Users.findOneAndUpdate { success: true, message: "Movie removed from favorites successfully" }
DELETE /users/:Username Username (string), MovieID (string) yes Allows users to delete user. updatedUser = await Users.findOneAndDelete { success: true, message: "User was deleted" }
