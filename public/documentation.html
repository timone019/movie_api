Movie and User API Documentation via Mongoose

Method Endpoint Parameters Request Body Description Mongoose Logic Expected Response
GET /movies Returns a list of all movies. Movie.find({}) [{ movieObject1 }, { movieObject2 }, ...]
GET /movies/:title Title (string) Returns data about a single movie. Movie.findOne({ title: req.params.title }) { movieObject }
GET /genres/:name Name (string) Returns data about a genre. Genre.findOne({ name: req.params.name }) { genreObject }
GET /directors/:name Name (string) Returns data about a director. Director.findOne({ name: req.params.name }) { directorObject }
POST /users/register { username, password, email, dob } Allows new users to register. User.create(req.body) { success: true, message: "User registered successfully" }
PUT /users/:userId UserID (string) { username, password, email, dob } Allows users to update their user info. User.findByIdAndUpdate(req.params.userId, req.body, { new: true }) { success: true, message: "User updated successfully", updatedUser: { updatedUserInfo } }
POST /users/:userId/favorites UserID (string) { movieId } Allows users to add a movie to their favorites. User.findByIdAndUpdate(req.params.userId, { $push: { favorites: req.body.movieId } }) { success: true, message: "Movie added to favorites successfully" }
DELETE /users/:userId/favorites/:movieId UserID (string), MovieID (string) Allows users to remove a movie from their favorites. User.findByIdAndUpdate(req.params.userId, { $pull: { favorites: req.params.movieId } }) { success: true, message: "Movie removed from favorites successfully" }
