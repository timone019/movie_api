Movie and User API Documentation

Movies

GET /movies List of all movies and details
GET /movies/:title Specific movie title
GET /movies/:genre/genreName Details of movie genre name, description and year
GET /movies/:directors/directorName Movie director
Post /users/:id/:movieTitle Add movie to favorite list
DELETE /users/:id/:movieTitle Delete a specific movie

Users

POST /users Create a new user
PUT /users/:id' Update name of a specific user
DELETE /users/:id Delete a specific user
